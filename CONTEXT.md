# Blog

The blog publishes authored material to reader-facing web, feed, and export surfaces.

## Language

**Published Content**:
Reader-facing material authored in this repository: Blog Posts, Talk Events, Code Projects, and Static Pages.
_Avoid_: Entry, item, document

**Content Catalog**:
The complete, validated set of Published Content in the repository at a point in time.
_Avoid_: Content files, markdown data

**Public Content Cache**:
The temporary reader-facing snapshot of Published Content governed by one freshness policy.
_Avoid_: Route cache, tag cache

**Blog Post**:
A dated article with a title, summary, body, and optional image.
_Avoid_: Blog entry, post record

**Talk Event**:
A dated talk or appearance with a title, summary, body, and optional image.
_Avoid_: Talk post, event record

**Code Project**:
A dated project with a title, destination, body, and publication state.
_Avoid_: Code entry, project record

**Static Page**:
An undated named page with a body and publication state.
_Avoid_: Page record, content page
